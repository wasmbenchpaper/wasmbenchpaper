# Python WASM SIMD Mandelbrot Set Benchmark

Prints the Mandelbrot Set using Python (Pyodide interpreter).

## Prerequisites

- NodeJS 18 (Use https://github.com/nvm-sh/nvm and `nvm install 18`)

## Steps to run

1. Install dependencies
    ```console
    $ make install
    ```
1. Run the NodeJS script
    ```console
    $ WITH_SIMD=true make run
    ```
1. Optional: You can change the number of rows, columns and iterations
    ```console
    $ WITH_SIMD=true NUM_ROWS=32 NUM_COLS=64 NUM_ITERS=50 make run
    ```
